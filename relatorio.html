Este site tem como objetivos a venda de produtos e notícias relacionadas ao gênero do rock. Tendo em vista o público alvo de pessoas que gostam de rock, todos os outros públicos são sempre bem vindos, neste site foram utilizadas várias tecnologias, tais elas como: Programação em Linguagem HTML, CSS e JavaScript, além de ferramentas que auxiliaram no planejamento e execução, como Docs Google, o Wareframe Moqups, Github e Discord.

Restrições e requisitos mínimos: navegadores web com tamanho de tela igual horizontal superior à: 320px ou igual ou inferior à 2560px.

Dificuldades encontradas: algumas aplicações de css, e javascript, como a criação de um carrossel de cards. Pontos fortes: Site interativo, com diversas abas, playlists e bandas. Pontos fracos: Foi um site feito por 3 alunos iniciantes na programação web, então ele não será um site aparentemente profissional

Periodicidade de atualização: semanalmente
